Perguntas Frequentes

Possui requisitos para adotar?

Sim. Nas Feiras de adoção é necessário assinar um termo de adoção que possui validade jurídica, o termo garante que o tutor vai levar o animal para casa. Além disso, a ONG vai realizar uma série de perguntas para analisar se a pessoa está em condições necessárias para adotar.

Possui requisitos para ser voluntário?

Sim. Ter disposição de tempo aos finais de semana e comprometimento com os horários e datas.

Todos os animais são castrados e foram devidamente medicados com vermífugo?

Sim. Todos os animais são castrados e devidamente medicados. Para os filhotes, a ONG espera até a idade segura para a castração.

Como posso auxiliar com o funcionamento da ONG?

É possível realizar uma transferência por meio do QRcode ou chave Pix no topo do site. Além disso, é possível realizar doações de cobertores, rações (na validade), comedouros, produtos de limpeza e outros itens.
