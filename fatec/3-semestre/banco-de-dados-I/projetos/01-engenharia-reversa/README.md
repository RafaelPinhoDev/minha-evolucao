# PROJETO 01 - ENGENHARIA REVERSA

## 🎯 Objetivo do Projeto

Desenvolvimento de um banco de dados relacional com integração entre tabelas por meio de chaves estrangeiras (Foreign Keys).

A partir de um diagrama de banco de dados previamente fornecido, foi realizada a reconstrução da estrutura utilizando a linguagem Transact-SQL (T-SQL), garantindo a correta modelagem e relacionamento entre as tabelas.
