# Projeto base para aula de Acessibilidade

## 🧠 Lógica de implementação

- Criação da aba de acessibilidade acima do menu de navegação com as seguintes funções:

* Botão de aumentar e diminuir a fonte
* Botão de aumentar e diminuir contraste
* Botão de mudar para o Tema Claro e Tema Escuro

## ⚠ Importante

O site vem sendo desenvolvido e aprimorado a cada aula da matéria de Acessibilidade com o objetivo de promover a acessibilidade e a inclusão para todas as pessoas.
